Hash

Mengubah teks menjadi hash SHA-256. Hash berubah saat input diubah.

Block

Melatih proses mining untuk mencari nonce dan membuat hash block.

BlockChain

Menambah block baru yang saling terhubung melalui previous hash.

ECC

Membuat key pair dan signature untuk verifikasi digital.

Konsensus

Simulasi transaksi antar pengguna dan pembentukan block baru.
